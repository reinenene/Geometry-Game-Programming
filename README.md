# Projet Géométrie 2D – Game Programming

Petit projet en C++ pour manipuler des **points** et des **vecteurs 2D**.  
On peut faire des translations, rotations et homothéties pour voir comment bougent les points et les vecteurs dans un plan 2D.  


---

## Structure du projet

main.cpp
geometry/
├── point.h / point.cpp
├── vector.h / vector.cpp
└── utils.h


---

### Fichiers

- **main.cpp** → pour tester les fonctions  
- **point.h / point.cpp** → structure et fonctions pour les points  
- **vector.h / vector.cpp** → structure et fonctions pour les vecteurs  
- **utils.h** → fonctions pratiques pour afficher les points/vecteurs facilement
